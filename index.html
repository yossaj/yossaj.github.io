Book Trip

This was a project made with Node.js, express, Leaflet.js, CSS Grid and the google books API.

Book Trip was designed to help user gain greater insight into the lives of various select authors throughout the world. It uses LeafletJS and the Google Books API to give an overview of the books each author wrote, where they lived throughout their lives and where some of their books were written

Dino-Hub

An educational app about dinosaurs.

The brief for this project was to create an educational app which encouraged children to learn about dinosaurs.

We wanted to make the children using the app feel as if they were taking a journey as they learned about dinosaurs. We did this by including an map which showed the dig sites where the dinosaurs was first found, a talking dinosaur kids can give a voice too, as well as locations where they can attend exhibitions to see the dinosaurs in person.

I had a great time working with my group on this and am looking forward to working with them again on future projects.
